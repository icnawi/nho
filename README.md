## How to run locally
1. This repo's default package manager: PnPM. To install it globally use npm: `npm i -g pnpm`
2. Check whether it works: `pnpm -v`
3. `pnpm i` to install deps
4. Run `pnpm copy:env` to prepare env
5. Modify `.env` as needed
6. `pnpm dev`
